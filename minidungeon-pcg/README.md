## Prerequisites
1. Install poetry

## Run Project
1. cd /game-ai-pcg/minidungeon-pcg
2. install poetry
3. poetry run start

## Run game
poetry run start ga_generated --env md-pygame --batch_size 10 --plays_per_level 10
