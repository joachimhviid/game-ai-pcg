# MiniDungeons PCG

## Prerequisites

1. Python 3.13 installed
2. Poetry installed (e.g. `pipx install poetry`)

## Run Project

1. Activate the virtual environment
    - Bash/Zsh/Csh: `eval $(poetry env activate)`
    - Powershell: `Invoke-Expression (poetry env activate)`
2. `poetry install`
3. `poetry run start <name-of-stage>` (Included stage is "pcg")

## Information

T = CHEST  
. = WALKING TILE  
\# = NON-WALKING TILE  
M = MONSTER  
S = SPAWN  
P = POTION  
E = EXIT/STAIRS
